# ESLint Config Generator

## Overview
Generate ESLint configurations with best practices for TypeScript, React, and Node.js projects

## Installation
```bash
npm install -g eslint-config-generator
```

## Usage
```bash
eslint-config-generator --init
eslint-config-generator --config ./config.json
eslint-config-generator --output ./dist
```

## Configuration
```json
{
  "preset": "recommended",
  "rules": {},
  "output": "./dist"
}
```
