# Prettier Formatter

## Overview
Automated code formatting tool with opinionated defaults for consistent code style

## Installation
```bash
npm install -g prettier-formatter
```

## Usage
```bash
prettier-formatter --init
prettier-formatter --config ./config.json
prettier-formatter --output ./dist
```

## Configuration
```json
{
  "preset": "recommended",
  "rules": {},
  "output": "./dist"
}
```
