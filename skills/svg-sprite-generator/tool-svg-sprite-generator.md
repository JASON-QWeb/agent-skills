# SVG Sprite Generator

## Overview
Generate optimized SVG sprite sheets from individual SVG files for efficient icon usage

## Installation
```bash
npm install -g svg-sprite-generator
```

## Usage
```bash
svg-sprite-generator --init
svg-sprite-generator --config ./config.json
svg-sprite-generator --output ./dist
```

## Configuration
```json
{
  "preset": "recommended",
  "rules": {},
  "output": "./dist"
}
```
