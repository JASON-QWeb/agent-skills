# TypeScript Compiler Helper

## Overview
TypeScript compilation helper with watch mode, incremental builds, and error reporting

## Installation
```bash
npm install -g typescript-compiler-helper
```

## Usage
```bash
typescript-compiler-helper --init
typescript-compiler-helper --config ./config.json
typescript-compiler-helper --output ./dist
```

## Configuration
```json
{
  "preset": "recommended",
  "rules": {},
  "output": "./dist"
}
```
