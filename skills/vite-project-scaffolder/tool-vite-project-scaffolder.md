# Vite Project Scaffolder

## Overview
Scaffold new Vite projects with pre-configured templates for React, Vue, and Svelte

## Installation
```bash
npm install -g vite-project-scaffolder
```

## Usage
```bash
vite-project-scaffolder --init
vite-project-scaffolder --config ./config.json
vite-project-scaffolder --output ./dist
```

## Configuration
```json
{
  "preset": "recommended",
  "rules": {},
  "output": "./dist"
}
```
